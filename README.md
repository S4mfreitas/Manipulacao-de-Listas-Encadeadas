# Manipulacao-de-Listas-Encadeadas
Exercícios de AEDS II (UNIMONTES) sobre Listas Encadeadas. Implementação de funções para concatenar duas listas, verificar igualdade de conteúdo (strings) e criar cópias independentes (deep copy) da estrutura. O código foca na manipulação correta de ponteiros e alocação dinâmica de memória.
